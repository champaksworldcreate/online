Pseudocode ‚Äî (Simple example: "Mini Insertion Sort" for small arrays)

We'll do a whiteboard-style dry run on the sample array and annotate each swap step live.

Code ‚Äî Implementations

Python

def mini_insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr

# Example:
print(mini_insertion_sort([5,2,7,3]))

You can copy-paste any snippet and run it during the live demo. We'll run the JS in the browser console for immediate visual feedback.

Complexity & Notes

  • Time complexity: O(n¬≤) worst-case, O(n) best-case (already sorted).
  • Space complexity: O(1) (in-place).
  • Good for: small arrays, almost-sorted data, teaching/learning steps.
